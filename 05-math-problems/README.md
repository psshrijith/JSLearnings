# Math Problems

This directory contains mathematical problems and numerical algorithm implementations in JavaScript.

## Topics Covered

### Basic Mathematical Operations
- **Factorial.js** - Calculate factorial of a number
- **Fibonacci.js** - Generate Fibonacci sequence
- **SumOf.js** - Calculate sum of numbers in a range
- **ExtractMaximum.js** - Extract maximum number from input

### Number Analysis
- **MaximumNumber.js** - Find maximum number in a collection
- **MinimumNumber.js** - Find minimum number in a collection
- **SecondLargest.js** - Find second largest number
- **kth_digit.js** - Extract kth digit from a number

### Prime Numbers
- **primeNumber.js** - Check if a number is prime
- **primeInRange.js** - Find prime numbers in a range
- **pairs_of_prime.js** - Find pairs of prime numbers

## Usage
Each file can be run independently with Node.js:
```bash
node filename.js
```

## Prerequisites
- Understanding of basic mathematics
- Knowledge of JavaScript number handling
- Node.js installed

## Learning Outcomes
- Master mathematical problem-solving techniques
- Understand number theory concepts
- Implement efficient numerical algorithms
- Improve analytical thinking for math problems
