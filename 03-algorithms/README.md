# Algorithms

This directory contains implementations of various algorithms and algorithmic patterns in JavaScript.

## Algorithm Categories

### Recursion
- **recursion/** - Recursive algorithm implementations
  - Basic recursion patterns
  - Divide and conquer algorithms
  - Tree and graph traversals

### Sliding Window
- **sliding-window/** - Sliding window technique implementations
  - Fixed-size sliding window
  - Variable-size sliding window
  - Maximum/minimum subarray problems

### Sorting
- **sorting/** - Various sorting algorithm implementations
  - Comparison-based sorting
  - Linear-time sorting
  - Custom sorting implementations

### Two Pointers
- **two-pointers/** - Two-pointer technique implementations
  - Array manipulation with two pointers
  - String processing with two pointers
  - Fast and slow pointer patterns

## Usage
Each file can be run independently with Node.js:
```bash
node filename.js
```

## Prerequisites
- Understanding of basic data structures
- Knowledge of time and space complexity
- Node.js installed

## Learning Outcomes
- Master fundamental algorithmic techniques
- Understand when to apply specific algorithms
- Improve problem-solving skills
- Learn to analyze algorithm efficiency
