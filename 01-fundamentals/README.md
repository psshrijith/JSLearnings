# JavaScript Fundamentals

This directory contains basic JavaScript concepts and fundamental programming exercises.

## Topics Covered

### Basic Concepts
- **HelloWorld.js** - Basic "Hello World" program
- **variables.js** - Variable declarations and scoping
- **dataTypes.js** - JavaScript data types and type checking
- **coercion.js** - Type coercion in JavaScript

### Functions
- **FunctionInJS.js** - Function declarations and expressions
- **firstclassfunctions.js** - Functions as first-class citizens
- **closures.js** - Closure concepts and examples

### Objects & OOP
- **objects.js** - Object creation and manipulation
- **oops1.js** - Basic OOP concepts
- **oops-2.js** - Advanced OOP concepts
- **oops-3.js** - Object-oriented programming patterns

### Arrays & Loops
- **Array.js** - Array operations and methods
- **loops.js** - Different types of loops in JavaScript
- **occurenceOfChar.js** - Character counting in strings
- **reverseString.js** - String reversal algorithms

### Asynchronous JavaScript
- **callback.js** - Callback functions
- **callbackhell.js** - Callback hell examples
- **promise.js** - Promise basics
- **promiseAll.js** - Promise.all() method
- **promiseallSettled.js** - Promise.allSettled() method
- **callbackhellpromises.js** - Converting callback hell to promises
- **asyncAwait.js** - Async/await syntax

## Usage
Each file can be run independently with Node.js:
```bash
node filename.js
```

## Prerequisites
- Node.js installed
- Basic understanding of programming concepts
