# String Manipulation

This directory contains exercises and implementations focused on string manipulation and processing in JavaScript.

## Topics Covered

### Basic String Operations
- **ReverseString.js** - Reverse a string
- **RemoveSpaces.js** - Remove spaces from a string
- **UpperCase.js** - Convert string to uppercase
- **Truncate.js** - Truncate string to specified length

### String Analysis
- **CountOfLetters.js** - Count occurrences of each letter
- **DuplicatesInString.js** - Find duplicate characters in string
- **NumbersInString.js** - Extract numbers from a string
- **UncommonCharacters.js** - Find uncommon characters between strings

### String Searching & Patterns
- **Anagram.js** - Check if two strings are anagrams
- **AnagramPalindrome.js** - Check for palindrome anagrams
- **Haystack.js** - Needle in haystack problem
- **ImplementStrStr.js** - Implement strStr() function
- **Pattern.js** - Pattern matching in strings

### String Transformation
- **Camelize.js** - Convert string to camelCase
- **MergeCharacters.js** - Merge characters from strings
- **MergeString.js** - Merge two strings
- **ReverseWords.js** - Reverse words in a string

### Advanced String Problems
- **LongestSubstring.js** - Find longest substring without repeating characters
- **LongestWithK.js** - Longest substring with K distinct characters
- **CheckSpam.js** - Check if text contains spam words
- **SaveIronman.js** - Ironman string problem

## Usage
Each file can be run independently with Node.js:
```bash
node filename.js
```

## Prerequisites
- Understanding of JavaScript strings
- Basic knowledge of string methods
- Node.js installed

## Learning Outcomes
- Master string manipulation techniques
- Understand string algorithms and patterns
- Improve problem-solving skills for string-based challenges
- Learn efficient string processing methods
